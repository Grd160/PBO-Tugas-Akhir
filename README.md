# 🎮 Shooter Game - PBO Project

Game platformer shooter 2D bergaya klasik (Run and Gun) yang dikembangkan menggunakan Python dan Pygame. Proyek ini merupakan tugas akhir mata kuliah Pemrograman Berorientasi Objek (PBO) untuk mengintegrasikan teori OOP ke dalam aplikasi interaktif.

# Fitur Utama
Sistem Stage Dinamis: Berbagai rintangan platform dan tata letak musuh yang berbeda di setiap stage.
Level Up & Upgrade: Pemain bisa naik level dengan mengalahkan musuh dan mendapat *item upgrade*.
Variasi Senjata: Terdapat berbagai jenis senjata mulai dari Standar, Rapid (senapan mesin), Laser (menembus objek), hingga Granat (ledakan).
Boss Fight: Pertarungan melawan bos di akhir map.

2. Buka terminal/Command Prompt di dalam folder proyek.
